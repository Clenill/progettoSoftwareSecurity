# Utilizzo
1. Installare [Docker](https://www.docker.com/) e [Docker Compose](https://docs.docker.com/compose/).
2. Creare il file `.env` partendo dal file `.env.example` presente nella repository.
3. Per avviare l'esecuzione, eseguire `docker compose up --build` nel terminale con permessi di superuser, posizionandosi nella directory di questo repository.
4. Per fermare l'esecuzione, eseguire `docker compose down` nel terminale con permessi di superuser, posizionandosi nella directory di questo repository.
